# Getting Started

Get started with Ammolite in the project.

## Installation

Install this package as a dev dependency in the project:

<PackageManagerTabs 
    command={{
        npm: "npm i ammolite",
        yarn: "yarn add ammolite",
        pnpm: "pnpm add ammolite",
        bun: "bun add ammolite",
    }} 
/>
